# Projeto-Integrador-III-B
Este projeto tem por objetivo otimizar a gestão dos colaboradores de uma rede de lava rápido, trazendo facilidade entre gestores e funcionarios, e a empresa com o cliente.

Esponjão OS - Sistema de Gestão de Estética Automotiva

O Esponjão OS é uma solução tecnológica desenvolvida para transformar a gestão operacional de lava-jatos e centros de estética automotiva. Através de uma interface moderna e funcional, o sistema permite o controle total de ordens de serviço, desde a entrada do veículo até a entrega final, utilizando metodologias ágeis de acompanhamento.🎯 Objetivo do ProjetoO objetivo central é resolver problemas comuns de pequenos negócios, como a falta de controle de prazos, esquecimento de tarefas e dificuldade na visualização do fluxo de trabalho. O sistema atua como um facilitador entre a equipe técnica e a gerência, garantindo organização e transparência nos processos internos.🚀 Funcionalidades PrincipaisPara garantir uma nota máxima em critérios técnicos e de usabilidade, o sistema contempla:Landing Page Institucional: Tela inicial voltada ao cliente, apresentando os serviços e facilitando o contato via WhatsApp.Autenticação Restrita: Sistema de login seguro para administradores, garantindo que apenas pessoal autorizado gerencie os dados.Quadro Kanban (Essencial): Gerenciamento visual dividido em colunas: "A Fazer", "Em Lavagem" e "Concluído".Gestão de Ordens de Serviço: Cadastro detalhado de veículos (placa, modelo), tipo de lavagem e definição de prazos.Integração com Calendário: Visualização dinâmica de todas as entregas programadas para o mês/semana.Painel de Relatórios: Dashboards com estatísticas em tempo real sobre a produtividade e volume de serviços.🛠️ Tecnologias UtilizadasO projeto foi construído utilizando uma arquitetura monolítica moderna, priorizando performance e facilidade de deploy:HTML5 & CSS3: Estrutura semântica e estilização avançada com variáveis globais e Dark Mode.JavaScript (Vanilla): Lógica de aplicação, manipulação de DOM e gerenciamento de estado.FullCalendar API: Biblioteca profissional para a renderização do módulo de agenda.FontAwesome: Conjunto de ícones vetoriais para interface intuitiva.Local Storage: Persistência de dados local, permitindo o uso do sistema sem dependência imediata de um banco de dados externo.📋 Metodologia de DesenvolvimentoO desenvolvimento seguiu as melhores práticas de Engenharia de Software:Levantamento de Requisitos: Identificação das dores reais de um lava-jato (esquecimento de modelos, confusão de prazos).Prototipagem: Design focado na experiência do usuário (UX), garantindo que o operador possa atualizar o status com poucos cliques.Desenvolvimento Ágil: Implementação iterativa com foco no MVP (Mínimo Produto Viável)
